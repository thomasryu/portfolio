# Portfolio

<br/>

My porfolio, built with Next.js, Tailwind CSS, and TypeScript, <br/>
hosted on Vercel.

### A live version can be found [here](https://www.thomasryu.com).

<br/>

## Setup

```bash
# Development
yarn dev

# Production
yarn build
yarn start
```

<small>The result will be shown at [http://localhost:3000](http://localhost:3000).</small>
